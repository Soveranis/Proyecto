¿Que tanto sabes de filosofia?

Guees the philosopher es un juego fácil de jugar donde pondrás a prueba tus conocimientos sobre los filósofos. Consiste en adivinar a qué filósofo se refiere cada pregunta.


Intrucciones para jugar

  • Inicia sesión para crear tu usuario y contraseña.
  • Este juego está recomendado para mayores de 6 años.
  • Tendrás un tiempo límite de 1 minuto para responder cada pregunta
  • Cada vez que respondas correctamente, subirás de nivel.
